# Adventures

Site de viagens com design **dark luxury editorial**, desenvolvido como projeto acadêmico do Professor Flores com HTML, CSS e JavaScript puro.

## Tema
Guia de viagens inspiracional  destinos de praia, montanha, cidade e deserto.

## Funcionalidades implementadas

| # | Funcionalidade | Tecnologia |
|---|----------------|------------|
| 1 | Menu responsivo com hamburguer (mobile) | JS + CSS |
| 2 | Link ativo no menu via scroll (IntersectionObserver) | JS |
| 3 | Animação de entrada dos cards ao rolar a página | JS (IntersectionObserver) |
| 4 | Sistema de busca de destinos | JS |
| 5 | Formulário de contato com validação em tempo real | JS |
| 6 | Botão "Voltar ao topo" | JS |
| 7 | Design responsivo (desktop / tablet / mobile) | CSS Media Queries |
| 8 | Layout com CSS Grid e Flexbox | CSS |

## Tecnologias utilizadas
- **HTML5** – estrutura semântica (header, main, section, article, footer, nav)
- **CSS3** – variáveis CSS, Flexbox, Grid, Media Queries, animações
- **JavaScript ES6+** – puro, sem frameworks
- **Google Fonts** – Playfair Display + DM Sans

## Acessibilidade (WCAG)
Atributos `alt` em todas as imagens
Labels em todos os campos de formulário
 `aria-label`, `aria-expanded`, `aria-live`, `aria-invalid`
 Navegação por teclado (foco visível, Enter/Space nos cards)
Contraste adequado entre texto e fundo

## Estrutura de arquivos
```
index.html   → estrutura semântica da página
style.css    → toda a estilização e responsividade
script.js    → interatividade e dinamismo
README.md    → este arquivo
```

## Como rodar
Abra o `index.html` diretamente no navegador — não requer servidor.
