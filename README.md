Adventurous

Site de viagens desenvolvido para a disciplina de Desenvolvimento Web.

Tema
Plataforma de exploração de destinos turísticos e promoções de viagem.

Tecnologias utilizadas
- HTML5 (estrutura semântica)
- CSS3 (estilização e responsividade)
- JavaScript puro (interatividade)

Funcionalidades
- Sistema de busca de destinos
- Efeitos visuais em promoções
- Rolagem suave entre seções

## Responsividade
O site se adapta para desktop, tablet e dispositivos móveis.

Acessibilidade
- Uso de alt em imagens
- Estrutura HTML semântica
- Navegação por teclado
